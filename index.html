Here is a paragraph element, a fundamental way to group text into a unit. The width it is allowed determines the number of lines that appear if no line break is forced.

To initiate a second paragraph, the first <p> element is closed and a new one is created. This is the second one. To represent HTML tags within HTML, the Entity representation must be used. In this case an entity name was used (as opposed to an entity number).

To learn more about Entities, visit the W3 Schools page.
