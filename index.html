Welcome to The ElemenTest, where I hope to test out as many HTML elements as I can.

Here is a paragraph element, a fundamental way to group text into a unit. The width the element is allowed determines the number of lines that appear if no line break is forced. It is known as a block-level element, as when it is used it gets put on its own "tier" so to say, and exits out of a line if it was initiated from one, into its own "block".

To initiate a second paragraph, the first <p> element is closed and a new one is created. This is the second one. To represent HTML tags within HTML, the Entity representation must be used. In this case an entity name was used (as opposed to an entity number). An interesting exercise that I should carry out later would be to alter the ordinal number word instances above (e.g., second) to match the nth <p> element, so should another paragraph get added before this one, the reference would still be correct.

To learn more about Entities, visit the W3 Schools page. The W3 Schools is a great reference, but not comprehensive enough to be used on its own. It should be supplemented by resources such as the Mozilla Developer Network (MDN) to add the lacking depth. The highlighting and links seen above are courtesy of inline-level elements. They exist, as their name indicates, inline wih the text they sit beside. Unlike the block-level elements, they don't create new lines just by existing, but are meant to alter the lines they reside in. The benefits of uninterrupted silence?

  • Peace. You can't put a price on that.
  • Clearer thoughts.
  • Increased productivity.
  • Getting in touch with oneself.
  • As stated above, no (well, to be fair, less) interruptions.

The downsides of uninterrupted silence?

  • The need to mask tinnitus.
  • Clearer thoughts -- Yes it's a double edged sword. Sometimes we use interruptions to avoid reflecting on difficult memories.
  • Losing track of time. Perhaps this should also be added to the benefits list though?
  • That uneasy feeling that every slight noise brings.
